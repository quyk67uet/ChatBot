# Chatbot Website with Streamlit and FastAPI

This project consists of a website with two chatbots:
1. **Chatbot for General Conversation**: A simple conversational chatbot.
2. **Vision Chatbot**: A chatbot that analyzes images uploaded by the user.

The project is structured into two main directories:
- `streamlit`: Contains the code for the Streamlit app (Vision Chatbot).
- `fastapi`: Contains the code for the FastAPI app (General Conversation Chatbot).

Both applications are containerized using Docker and managed with Docker Compose for easy deployment.

## Prerequisites

- Docker: [Install Docker](https://docs.docker.com/get-docker/)
- Docker Compose: [Install Docker Compose](https://docs.docker.com/compose/install/)
